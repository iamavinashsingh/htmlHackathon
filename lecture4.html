üåê Why Forms?

Forms web ko read-only se interactive banata hai. Forms ke bina:

  • Login / Signup nahi ho paata
  • Search / Comments / Checkout possible nahi hota
  • Server ko data bhejna mushkil ho jaata

üëâ Forms = User input collect + Server ko bhejna

üèó Core Structure

 
<form action="server-endpoint" method="POST">
  <!-- Fields -->
  <input type="text" name="username" placeholder="Enter Name">
  <button type="submit">Submit</button>
</form>

Form Attributes:

Attribute Purpose Example Values
action Data kahan bhejna hai (server endpoint) "/submit-form", "process.php"
method Kaise bhejna hai (GET ya POST) GET (default), POST
target Response kahan display karna hai _blank, _self, _parent, _top
autocomplete Autocomplete enable/disable karta hai on, off

‚ú® Building Blocks - Input Elements

The <input> element is the most important form element. It can be displayed in many ways, depending on the type attribute.

<input type="text" name="username" placeholder="Enter your username">

Common Input Attributes:

Attribute Purpose Example
type Specifies the input type text, password, checkbox, etc.
name Input ka naam (form submission ke liye important) username, email, password
value Default value specify karta hai value="default text"
placeholder Expected value ka hint provide karta hai placeholder="Enter your name"
required Specifies that the input must be filled out required

üìÑ Textarea Element

The <textarea> element defines a multi-line text input control:

HTML Code

<label for="comments">Special Instructions:</label><br>
<textarea id="comments" name="comments" rows="4" cols="40"></textarea>

Browser Output

Textarea Attributes:

Attribute Purpose Example
rows Specifies the visible number of lines rows="4"
cols Specifies the visible width cols="50"
placeholder Provides a hint about the expected value placeholder="Enter your message here"

üîΩ Select Dropdown

The <select> element defines a drop-down list:

HTML Code

<label for="country">Country:</label><br>
<select id="country" name="country">
  <option value="">--Select--</option>
  <option value="in">India</option>
  <option value="us">USA</option>
  <option value="uk">UK</option>
</select>

Browser Output

Multiple Selection

Allow multiple selections with the multiple attribute:

HTML Code

<select multiple>
  <option value="html">HTML</option>
  <option value="css">CSS</option>
  <option value="js">JavaScript</option>
  <option value="python">Python</option>
</select>
