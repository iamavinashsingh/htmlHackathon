üè∑Ô∏è HTML Headings

Purpose ‚Üí Headings se aap apne webpage ka structure aur outline banate ho (jaise book ke chapters aur sub-chapters).

Range ‚Üí <h1> sabse bada aur important hota hai ‚Üí <h6> sabse chhota aur least important.

Tag Purpose Example

<h4> Aur chhoti detailing Sub-points under advantages
<h5> & <h6> Bahut hi minor headings (rarely used) Fine details or legal text headings

üìÑ The Paragraph Tag
  • Purpose ‚Üí Webpage me text ko proper paragraphs me divide karne ke liye use hota hai.
  • Syntax ‚Üí <p>...your text here...</p>
  • Default Behavior ‚Üí Browser automatically ek line break aur thoda space paragraph ke upar aur niche add karta hai.

Best Practice

  • Text ke alag-alag blocks ko separate karne ke liye multiple <p> tags use karo.
  • Kabhi bhi extra line breaks (<br>) daal ke paragraph banane ki galti mat karna.

Example

HTML Code Browser Output

<p>This is the first paragraph. It contains multiple sentences to demonstrate how paragraph tags work in HTML.</p>

<p>This is the second paragraph. Notice how browsers automatically add space between paragraphs.</p>

This is the first paragraph. It contains multiple sentences to demonstrate how paragraph tags work in HTML.

‚ûñ Horizontal Rule
  • Purpose ‚Üí Webpage me sections ko visually separate karne ke liye horizontal line add karta hai.
  • Syntax ‚Üí <hr>
  • Self-closing Tag ‚Üí Isme closing tag nahi hota (jaise <br>).
  • Default Behavior ‚Üí Browser ek horizontal line draw karta hai jo poore width me hoti hai.

Best Practice

Jab aapko content ke beech me clear visual break chahiye tab use karo.

‚Ü©Ô∏è Line Break
  • Purpose ‚Üí Text me line break insert karta hai bina naya paragraph banaye.
  • Syntax ‚Üí <br>
  • Self-closing Tag ‚Üí Isme closing tag nahi hota (jaise <hr>).

üîó The Anchor Tag
  • Purpose ‚Üí Webpage me hyperlinks create karta hai jo users ko ek page se dusre page ya website par le jaate hain.
  • Syntax ‚Üí <a href="URL" target="_blank" title="xyz">Clickable Text</a>

Attributes

Attribute Purpose Example Values
href jahan link point karega uska URL specify karta hai "page.html", "https://example.com"
target link ko kaise open karna hai specify karta hai _blank (new tab), _self (same tab)
title link ke baare me extra info provide karta hai jo hover karne par dikhai deti hai "Visit our homepage"

Best Practices

  • Descriptive link text use karo jo bataye ki link kya karta hai. Example: "Click here" ki jagah "Read more about HTML".
  • Agar link external site par ja raha hai to target="_blank" use karo taaki user ka current page open rahe.

Examples

HTML Code Browser Output

<a href="https://www.google.com" target="_blank" title="Visit Google">Go to Google</a>

<a href="about.html">About Us</a>

Go to Google

About Us

üñºÔ∏è The Image Tag
  • Purpose ‚Üí Webpage me images embed karne ke liye use hota hai.
  • Syntax ‚Üí <img src="imageURL" alt="description" width="value" height="value">

Attributes

Attribute Purpose Example
src image ka URL ya file path specify karta hai "image.jpg", "https://example.com/image.png"
alt image ka description provide karta hai "Photo of MS Dhoni"
width & height image ke dimensions specify karte hain pixels me width="400", height="300"

Best Practices

  • Har image ke liye meaningful alt text provide karo taaki accessibility improve ho.
  • Image size optimize karo taaki webpage fast load ho (large images se page slow ho sakta hai).
  • Agar specific size chahiye to width aur height attributes use karo, warna browser original image size use karega.

Example

HTML Code Browser Output

<img src="https://upload.wikimedia.org/wikipedia/commons/d/d5/MS_Dhoni_%28Prabhav_%2723_-_RiGI_2023%29.jpg" alt="MS Dhoni" width="400" height="400">
